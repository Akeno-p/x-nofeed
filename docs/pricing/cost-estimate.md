## 他人のポスト表示機能を追加した場合のAPI利用コスト試算

※ 投稿の読み取り: $0.005/件 で計算（$1 = ¥150換算）
※ ユーザー情報の読み取り（$0.010/件）は含めていない。投稿取得時にexpansionsで付随するユーザー情報が「投稿の読み取り」に含まれるのか「ユーザーの読み取り」として別途課金されるのかは公式ドキュメントに明記がないため、実装時にダッシュボードで要確認。

### パターン1: 投稿のみ取得

| 項目       | 数量/日             | 日額  | 月額                  |
| ---------- | ------------------- | ----- | --------------------- |
| 投稿の取得 | 150件（50人×3投稿） | $0.75 | $22.5                 |
| **合計**   |                     |       | **$22.5（約¥3,375）** |

### パターン2: 投稿 + リプライも取得

| 項目           | 数量/日                     | 日額   | 月額                    |
| -------------- | --------------------------- | ------ | ----------------------- |
| 投稿の取得     | 150件（50人×3投稿）         | $0.75  | $22.5                   |
| リプライの取得 | 2,000件（20投稿×平均100件） | $10.00 | $300                    |
| **合計**       |                             |        | **$322.5（約¥48,375）** |
